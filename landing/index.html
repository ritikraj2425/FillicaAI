Open-Source Security Notice

Fillica is a free, non-profit open-source project. Because it is not digitally signed by a paid Apple/Microsoft developer account, your OS may show a "damaged" or "unrecognized app" warning. You have two ways to run it:

Option 1: Clone Locally

Clone the GitHub repository and run npm install & npm run dev to build it directly from source on your machine.

Option 2: Bypass Security

Mac: Move app to Applications, open Terminal, and run:
xattr -cr /Applications/Fillica.app

Windows: If SmartScreen blocks it, click "More info" → "Run anyway".
